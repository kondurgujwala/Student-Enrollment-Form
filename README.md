Student Enrollment Form using JsonPowerDB

A web-based Student Enrollment Management System developed using HTML, JavaScript, Bootstrap, AJAX, and JsonPowerDB. The application allows users to add, manage, and update student records efficiently through a simple and responsive interface.

Features:

* Student Enrollment Form
* Input Validation
* Save Student Records
* Update Existing Records
* Reset Form Functionality
* JsonPowerDB Integration
* Responsive User Interface

Database:

* Database Name: SCHOOL-DB
* Relation Name: STUDENT-TABLE
* Primary Key: Roll-No

Technologies Used:

* HTML5
* CSS3
* Bootstrap
* JavaScript
* jQuery
* AJAX
* JsonPowerDB

This project was developed as part of the Login2Xplore JsonPowerDB Learning Course.
